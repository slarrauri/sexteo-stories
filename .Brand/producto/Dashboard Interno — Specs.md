# 🖥️ Dashboard Admin — App Separada

> Esta funcionalidad será una aplicación independiente del frontend principal de Sexteo.

---

## Pantallas del Dashboard Admin

| Pantalla | Ruta | Descripción |
|---|---|---|
| **Home Analytics** | `/admin` | Historias activas, mensajes/min, química promedio, NHI score |
| **Métricas de Matching** | `/admin/matching` | Success rate, story depth, dropoff point, re-match probability |
| **Reportes** | `/admin/reports` | Ver reportes, suspender/banear, logs de chat reportados |
| **Mapa de Géneros** | `/admin/genres` | Heatmap actividad y retención por género |
| **Panel IA** | `/admin/ai` | Desempeño IA narrador: conversaciones salvadas, aceptación, satisfacción |
| **Retención** | `/admin/retention` | Cohortes de historias, engagement por nivel |
| **Riesgo Narrativo** | `/admin/risk` | Alertas: chats fríos, frustración, fatiga narrativa |

---

## Indicadores Clave

- 🔥 Historias activas ahora
- 💬 Mensajes por minuto
- ❤️ Nivel promedio de química narrativa
- ⏳ Duración media de sesiones
- 📈 Tasa de continuidad de historias

---

## Narrative Health Index (NHI)

| Score | Estado |
|---|---|
| 90+ | Ecosistema magnético |
| 70–89 | Saludable |
| 50–69 | Inestable |
| <50 | Riesgo crítico |

---

## Widgets Críticos

- Top historias vivas ahora
- Matches con mayor química en tiempo real
- Usuarios narrativamente influyentes
- Historias virales emergentes
