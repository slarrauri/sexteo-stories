# 🧠 Sistema de Matching Psicológico Narrativo

## 1. Objetivo

Emparejar usuarios que tengan **química narrativa**, no solo gustos similares.

👉 La compatibilidad se calcula según:

* estilo narrativo
* ritmo emocional
* fantasías temáticas
* límites personales
* intensidad deseada
* rol dominante en historia

---

## 2. Perfil Narrativo del Usuario

Cada usuario tiene un **Narrative DNA Profile** generado por onboarding + comportamiento.

### Campos base

* Estilo: poético / directo / descriptivo / minimalista
* Ritmo: lento / progresivo / intenso
* Tipo de rol: guía / reactivo / explorador
* Nivel de improvisación
* Preferencia de trama: romántica, oscura, misteriosa, etc.
* Nivel de iniciativa narrativa

---

## 3. Test Inicial Inteligente (Onboarding)

En vez de preguntas aburridas, se usa un mini-juego narrativo.

Ejemplo:

> “Entrás a una habitación desconocida. ¿Qué hacés?”

Opciones analizadas:

* acción elegida
* vocabulario
* longitud de respuesta
* emoción transmitida

La IA detecta:

* imaginación
* seguridad narrativa
* creatividad
* estilo emocional

---

## 4. Algoritmo de Compatibilidad

Score de 0 a 100 basado en 5 capas:

| Capa                      | Peso |
| ------------------------- | ---- |
| Compatibilidad de límites | 30%  |
| Química narrativa         | 25%  |
| Estilo literario          | 15%  |
| Ritmo emocional           | 15%  |
| Objetivo de historia      | 15%  |

---

## 5. Matching Dinámico (aprende con uso)

El sistema se entrena automáticamente:

* analiza chats exitosos
* detecta cuándo una historia se corta
* mide engagement narrativo
* aprende qué combinaciones funcionan

Resultado:
👉 cada match mejora con el tiempo

---

## 6. Tipos de Match

No todos los matches son iguales:

| Tipo         | Descripción                                    |
| ------------ | ---------------------------------------------- |
| Spark Match  | alta química instantánea                       |
| Slow Burn    | compatibilidad a largo plazo                   |
| Chaos Match  | historias impredecibles                        |
| Mirror Match | estilos casi idénticos                         |
| Opposites    | estilos opuestos que generan tensión narrativa |

---

## 7. Score Visible al Usuario

En vez de % aburrido:

Ejemplo:

> Compatibilidad narrativa: **Alquimia Alta**

Niveles:

* Neutral
* Intriga
* Conexión
* Tensión
* Alquimia
* Destino Narrativo

---

## 8. Feature Diferencial: Predicción de Historia

Antes de empezar chat:

> “Probabilidad de historia intensa: 87%”

Esto genera anticipación emocional.

---

## 9. Motor Antifricción

Si la conversación decae:

La IA interviene con:

* giro de trama
* evento inesperado
* personaje secundario
* desafío narrativo

---

## 10. Ventaja Competitiva Real

Las apps actuales emparejan personas.
Este sistema empareja **experiencias potenciales**.

Eso es extremadamente difícil de copiar porque requiere:

* dataset narrativo
* análisis semántico
* machine learning conductual
* diseño psicológico
